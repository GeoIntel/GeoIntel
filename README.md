# GeoIntel

Open source GEOINT tool.

## Let's Go

### Stable Release

```
mkdir GeoIntel
cd GeoIntel
mkdir ch_data
mkdir ch_logs
wget https://github.com/GeoIntel/GeoIntel/releases/download/v0.0.8/docker-compose.yml
docker compose up -d
```

### Main Branch

```
git clone https://github.com/GeoIntel/GeoIntel
cd GeoIntel
mkdir ch_data
mkdir ch_logs
docker compose up -d
```
